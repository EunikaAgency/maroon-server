Enhance your WooCommerce store with the Product Variations Styler! This plugin allows you to dynamically change the product thumbnails based on attributes like color and size, and redirects customers to a customized product page.

== Description ==

Product Variations Styler is a powerful plugin for WooCommerce that transforms the way customers view and interact with product variations. By clicking on attributes such as color and size in the product loop, customers are redirected to a customized product page, providing a seamless and intuitive shopping experience.

Key Features:
- Dynamically change product thumbnails in the WooCommerce loop.
- Clickable attributes for immediate visual feedback and redirection to a customized product page.
- Replace the default 'content-product.php' in your theme with the plugin's version to implement the feature.
- Add a filter to 'woocommerce_locate_template' for overriding WooCommerce template files with those in the plugin.
- Custom product page templates included within the plugin.
- Built-in SCSS compiler for advanced styling located in the assets folder.
- Built-in Blade template engine, with compiled files located in the ../app/views/template/ folder.

== Installation ==

1. Upload the plugin files to the '/wp-content/plugins/product-variations-styler' directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the Settings->Product Variations Styler screen to configure the plugin.

== Frequently Asked Questions ==

= Can I customize the look of the product thumbnails? =

Yes, the plugin includes a built-in SCSS compiler, allowing you to customize the style of the thumbnails as per your needs.

= Does this plugin work with all WordPress themes? =

Product Variations Styler is designed to work with most WordPress themes. However, for best compatibility, ensure your theme follows standard WooCommerce coding practices.
